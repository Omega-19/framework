NPM

npm(Node Package Manager) est un gestionnaire de paquets pour l'écosystème Node.js. Il est utilisé pour installer, mettre à jour et gérer les dépendances des projets Node.js. NPM permet aux développeurs de partager et de réutiliser des bibliothèques de code (appelées "paquets" ou "modules") développés par d'autres, ainsi que de publier leur propres paquets pour être utilisés par la communauté.

Les versions

Nous avons trois versions: la version majeure, mineure et patch

Version majeure est la première version créee qui peut necessité un changement radical au point de ne plus fonctionner avec l'application initiale

Version mineure est la deuxième créée qui ont de petit changement de plus mais continue de fonctionner avec la meme application

Version patch est la celle qui a subit qq modification mais fonctionne toujours.

TypeScript

TypeScript est un langage de programmation fortement typé qui s'appuie sur JavaScript, vous offrant de meilleurs outils à n'importe quelle échelle.

Pour installer TypeScript il faut entrer la commande npm i -g typescript dans le terminal avec le g pour dire de façon global.

RESUME DE TOUS

TypeScript est un langage de programmation open source développé par Microsoft. Il est conçu comme un sur-ensemble typé de JavaScript; ce qui signifie que tout code JavaScript valide est également du code TypeScript valide. Cependant, TypeScript offre des fonctionnalités supplémentaires qui améliorent le développement et la maintenance des applications JavaScript à grande échelle.

Les avantages et/ou fonctionnalités du TypeScript

    Types de variables TS

  1. -number: les nombres, qu'ils soient entiers ou à virgule
  2. - string: représente les chaînes de caractères
  3. -boolean: représente les valeurs boléennes, c-a-dire true or false
  4. -array: représente un tableau contenant des éléments du même type
  5. -object: représente un type "d'objet"
  6. -null: représente la valeur "null"
  7. -undefined: représente la valeur "undefined"
  8. -any: C'est un type de variable dont les formes sont acceptées par typescript.Il représente un type non spécifié ou dynamique.
  9. -unknown: représente un élément dont on ne connait pas le type
  10. -enum: représente un ensemble de valeurs nommées, énumérées
  11. -tuple: représente un tableau avec un nombre fixe d'éléments de types différents.

Les unions en TS

les classes en typescript

  • public: par défaut, il permet l'accès à la propriété de partout.
  • private: ne permet l'accès que dans la classe. Il nous permet de ne plus utiliser les barres de 8 ou dièse
  • protected: permet l'accès dans la classe et dans les enfants de la classe.

La déclaration d'une classe ne nécessite pas forcément les getters et setters mais à la place on peut avor une fonction get() qui retourne un élément privé
